Campo Minado
Este é um jogo simples de Campo Minado desenvolvido em Java. O jogo consiste em um tabuleiro com células ocultas, algumas das quais contêm minas. O objetivo do jogo é abrir todas as células que não contêm minas sem detonar nenhuma das células que contêm minas.
Tecnologias Utilizadas:
Java: A linguagem de programação principal utilizada para desenvolver o jogo.
Swing: O framework gráfico utilizado para criar a interface do usuário.
